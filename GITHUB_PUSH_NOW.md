## ⚡ FEHLERBEHANDLUNG

### Problem: "Permission denied (publickey)"

```bash
# SSH Key nicht konfiguriert
# Nutze stattdessen HTTPS:

git remote set-url origin https://github.com/beastwareteam/OneClickRGB.git

# Oder nutze GitHub CLI:
gh auth login
```

### Problem: "Everything up-to-date"

```bash
# Repository ist bereits synchronized
# Das ist OK! Kontrolliere auf GitHub dass Dateien da sind
```

### Problem: "LFS file(s) detected"

```bash
# GitHub Large File Storage ist nicht enabled
# Für dein Projekt brauchst du das nicht

# Falls du große Dateien hast:
# 1. Entferne sie: git rm large-file.zip
# 2. Oder: git lfs install && git lfs track "*.zip"
```
